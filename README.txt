Requisitos

- node - v14.15


Detalhes

Para testar é necessário alterar duas listas, uma de objetos com preço e quantidade:
var items = [{ price: 25, qtd: 2 }, { price: 50, qtd: 1 }]

E outra com uma lista de string os e-mails:
var emails = ['ju1@g.com', 'ju2@g.com', 'ju3@g.com']

O preços devem ser em centavos e valores inteiros. 
As quantidade devem ser em valores inteiros.
A função valida e-mails repetidos, totais negativos e listas vazias.


Como executar: 

node index.js
